## 🚀 Como Rodar os Códigos

```bash
# Clone o repositório
git clone https://github.com/joaogfc/UFMG.git

# Acesse a pasta de interesse
cd UFMG/Semana\ 3\ -\ Variáveis

# Compile com gcc (ou outro compilador de sua preferência)
gcc arquivo.c -o programa

# Execute
./programa
```
